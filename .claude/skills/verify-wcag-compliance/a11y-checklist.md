# WCAG 2.1 AA Compliance Checklist

## Images
- [ ] Do all `<Image>` components have a non-empty `alt` prop (unless decorative)?
- [ ] Are decorative images hidden with `aria-hidden="true"`?

## Forms
- [ ] Do all inputs have an associated `<Label>`?
- [ ] Do error messages have `role="alert"`?

## Interactive
- [ ] Do all icon-only buttons have `aria-label`?
- [ ] Is the focus ring visible on all interactive elements?

## Structure
- [ ] Is there only one `<h1>` per page?
- [ ] Are heading levels sequential (h1 -> h2 -> h3)?
- [ ] Is there a "Skip to Content" link?
